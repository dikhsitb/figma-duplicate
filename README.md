# Linkd Landing Page

A marketing landing page for **Linkd**, built from a Figma design. The layout includes a navigation bar, hero section, feature grid, and alternating content blocks.

## Tech Stack

- **React** with **TypeScript**
- **Vite** for development and production builds
- **Tailwind CSS** for styling
- **Plus Jakarta Sans** via Google Fonts

## Design Source

This project was generated from a Figma frame using the **Figma MCP** integration in Cursor. The MCP server pulled design context (layout, typography, colors, and assets) from the source file, which was then adapted into a responsive React application.

**Figma file:** [4-levels--Copy-](https://www.figma.com/design/a1xsOhBxIdB451YlXk68y7/4-levels--Copy-?node-id=1-4250)

## Live Preview

Run locally with `npm install` followed by `npm run dev`.
